The Linking Project is an application that uses highly configurable matching criteria to perform deduplication and linking on datasets.

The Linking Project will help you:

  • Identity common entities in a CSV formatted dataset [de-duplication]

  • Identify common entities between two CSV formatted datasets [linking]

Datasets

Datasets are the source of data for the linking tool. To add a new dataset to the linking tool, you first must add your data file to the dataset folder of the linking application (files/media/datasets). Currently, only comma separated values (.csv) files are supported.

Projects

You can perform deduplication and linking on your datasets via projects. When you create a new project you will be prompted to pick either the deduplication or linking project.

A deduplication project finds common entities within a single dataset, whereas a linking project finds common entities between two datasets.
